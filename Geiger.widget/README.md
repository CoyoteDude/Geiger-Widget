# Geiger Lab — Übersicht Desktop Widget

Fully interactive Geiger counter / gamma spectrometer simulation that lives on your macOS desktop.

## Features
- 5 realistic lab isotopes (Cs-137, Co-60, Na-22, Ba-133, Am-241) with accurate gamma peaks
- Live updating Radiacode-style spectrum with labeled photopeaks
- CPM and µSv/h with natural statistical fluctuation
- Authentic Geiger clicks via Web Audio
- Intensity control, pause, reset spectrum, etc.

## Installation

1. Install [Übersicht](https://tracesof.net/uebersicht/)
2. Copy this entire `Geiger.widget` folder to:
   `~/Library/Application Support/Übersicht/widgets/`
3. Restart Übersicht or refresh widgets from the menu bar icon.
4. Right-click desktop → Edit Widgets and place the "Geiger Lab" widget.

**Recommended size:** 480 × 360 px or larger.

The widget is 100% interactive.

## Updating
Replace `geiger.html` with an updated version of the simulation and refresh widgets.

## Requirements
- macOS 12+
- Übersicht 1.6+
